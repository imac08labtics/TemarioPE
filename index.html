1.1 Conceptos basicos de estadistica: Se ocupa de organizar informacion y procesarla para deducir sus caracteristicas 1.2 Descripcion de datos: Organiza los datos de manera que se puedan observar mejor y entenderlo de una manera mas sencilla 1.3 Mediddas de tendencia central: Son valores que ayudan a visualizar el conjunto de datos y calculan esos datos 1.4 Parametros para datos agrupados: Son como etiquetas que resumen grandes grupos de informacion para poder observar mejor los datos dispersados 1.5 Distribucion de frecuencias: Organiza la informacion para ver cuantas veces se repite 1.6 Tecnicas de agrupacion de datos: Analiza la informacion y busca agruparlos datos mediante un metodo 1.7 Tecnicas de muestreo: Seleccionan cierta cantidad para estudiarla para no tardarse mas tiempo en estudiarlo todo completo
